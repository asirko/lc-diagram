À noter :

  • Les custom elements ne sont pas supporté en SVG, il faut donc faire un sélecteur par attribut (selector: '[attribute]') ;
  • Les attributs SVG n'ont pas d'équivalent en propriété, il faut donc utiliser ([attr.attribut]) ;
  • Le parser Angular ne reconnait pas directement les balises SVG dans les templates SVG, il faut leur ajoute le préfix (<svg:rect>) ;
